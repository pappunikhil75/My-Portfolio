My Portfolio Website

This is a personal portfolio website developed using HTML, CSS, and JavaScript. The purpose of this project is to showcase my technical skills, projects, and personal information in a professional way.

The website includes sections such as Home, About Me, Skills, Projects, and Contact, allowing visitors to easily learn more about my work and experience. It is designed with a clean and responsive layout so that it works well on different devices like desktops, tablets, and mobile phones.

This project demonstrates my knowledge of frontend web development, including page structure with HTML, styling with CSS, and basic interactivity using JavaScript.
